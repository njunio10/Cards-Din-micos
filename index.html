Sobre o Projeto

Este projeto celebra a história do Santos Futebol Clube por meio de uma experiência visual dinâmica. Utilize o menu superior para navegar entre as seções História, Títulos, Ídolos e Gols. Cada página apresenta cards interativos com imagens, vídeos e GIFs relacionados ao clube.

A bola giratória exibida abaixo foi integrada a partir de um código disponível no site CodePen, criado por jkantner, onde desenvolvedores compartilham projetos prontos e criativos.
